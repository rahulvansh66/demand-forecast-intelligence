# Data Analytics Skills Index

## How to Use These Skills

Since these skills are stored outside the Claude Code auto-discovery path, you'll need to manually reference them when needed. Use this format:

```
Use the skill from: external-skills/data-analytics-skills/[category]/[skill-name]/SKILL.md
```
